About Inkscape patch script

Introduction

UniConvertor is used as an external vector graphics translation tool by different projects and Inkscape included. UniConvertor project has been initiated at the request of Inkscape Team after Libre Graphics Meeting 2007. So we are continuing to support this integration.

Initially Inkscape has built-in copy of UniConvertor. But this copy is ancient and moreover it is broken. So we have created small script for resolving this issue. The script changes some files in Inkscape installation to switch using external stand-alone UniConvertor 1.1.5.
