About UniConvertor

General description

UniConvertor is a universal vector graphics translator. It is a command line tool which uses sK1 object model to convert one format to another. Application usage is an quite simple. For example:

c:\uniconvertor my_drawing.cdr my_drawing.pdf

Application is a multiplatform software. UniConvertor doesn't use external tools for vector graphics file translatio. So you can converting your drawing without installations of proprietary software like CorelDRAWTM or Adobe® IllustratorTM. Even on those platforms which are not supported by proprietary software vendors.

UnConvertor supports following vector graphics formats:

Import filters: Export filters:
  • CorelDRAW ver.7-X3,X4 (CDR/CDT/CCX/CDRX/CMX)
  • Adobe Illustrator up to 9 ver. (AI postscript based)
  • Postscript (PS)
  • Encapsulated Postscript (EPS)
  • Computer Graphics Metafile (CGM)
  • Windows Metafile (WMF)
  • XFIG
  • Scalable Vector Graphics (SVG)
  • Skencil/Sketch/sK1 (SK and SK1)
  • Acorn Draw (AFF)
  • HPGL for cutting plotter files(PLT)
  • Autocad Drawing Exchange Format (DXF)
  • DST design format (Tajima)
  • PES embroidery file format (Brother)
  • EXP embroidery file format (Melco)
  • PCS design format (Pfaff home)
  • AI (Adobe Illustrator ver.5.0 format)
  • SVG (Scalable Vector Graphics)
  • SK (Sketch/Skencil format)
  • SK1 (sK1 format)
  • CGM (Computer Graphics Metafile)
  • WMF (Windows Metafile)
  • PDF (Portable Document Format)
  • PS (PostScript)
  • PLT (HPGL for cutting plotter)

GUI frontend

Command line is a cool issue to be hardcore hacker but a lot of users preffering GUI application rather. So we have released GUI frontend for UniConvertor. Here are screenshots of UniConvertor in stand-alone mode:

In stand-alone mode application can be launched from system Start Menu or running command uniconvw.vbs

Using this mode you need choosing file, translation format and after that just to click "Convert" button. UniConvertor will save translation adding format extension to your file. For example, you have selected my_drawing.cdr. Resulted file name durin translation into WMF format will be my_drawing.cdr.wmf Such naming rule is used to avoid potential file clashig.

File manager integration

On Windows, Linux or MacOS X everyday you are working with files and file managers like Windows Explorer, Nautilus, Dolphin etc. are your day-by-day tools. So UniConvertor integration with file managers is a highly useful issue. In this release we have implemented this feature. Just click on file in your file manager and you find "Convert by UniConvertor" context menu option:

Selecting this option you launch UniConvertor with file reference:

Choice required translation format and press "Convert" button

After some time depending on file size and file format you will get translation result.
